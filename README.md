# Fallout 76 Season S.C.O.R.E. Calculator

![S.C.O.R.E. Logo](score-logo.svg)

This is a simple web-based calculator to help Fallout 76 players track their Season S.C.O.R.E. progression. It estimates how many daily and weekly challenges are needed to reach a target season rank.

Link: https://elia04ita.github.io/fallout76-score-calculator/

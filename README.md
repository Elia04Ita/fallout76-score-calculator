# Fallout 76 Season S.C.O.R.E. Calculator

This is a simple web-based calculator to help Fallout 76 players track their Season S.C.O.R.E. progression. It estimates how many daily and weekly challenges are needed to reach a target season rank, taking into account various S.C.O.R.E. boosters.
